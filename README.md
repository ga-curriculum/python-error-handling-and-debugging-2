## About
Practice identifying and fixing common errors in Python code, along with creating programs that can anticipate specific error types.

## Learning Objectives

*You will be able to:*

- Identify common errors in Python.
- Read error messages for guidance to fix errors.
- Implement `try`/`except` error handling.
